### 思路
记录插入元素的相对顺序，想象成在一维坐标轴上往原点两边进行插入，
使用哈希表记录插入位置
求解的时候只需要比较查询元素到两边的距离，取最小值即可

### 代码
```python
n = int(input())
dic = {}
l = 0
r = -1
for i in range(n):
    op, num = input().split()
    num = int(num)
    if op == "L":
        dic[num] = l - 1
        l -= 1
    elif op == "R":
        dic[num] = r + 1
        r += 1
    else:
        cur = dic[num]
        print(min(r - cur, cur - l))
```
### 复杂度分析
- T:O(n)
- S:O(n)
